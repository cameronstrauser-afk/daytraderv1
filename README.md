# Day Trading Stock Predictor

A Streamlit app with:
- BUY / HOLD / SELL prediction
- confidence %
- buy / hold / sell vote percentages
- top 5 strategies with reasons
- risk level
- paper trading mode
- watchlist scanner
- alerts
- short-selling toggle
- TradingView-inspired interface

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
